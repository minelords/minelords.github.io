## stash

> .git stash将本地的修改保存起来,并且将当前代码切换到HEAD提交上

```
git stash lish   #查看当前stash中的内容

git stash pop  #将当前stash中的内容弹出，并应用到当前分支对应的工作目录上,可以理解为恢复内容。
```

同样进入文件夹，输入`git stash pop`查看缓存内容
